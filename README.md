# Benkei2

Benkeiは、MacOSで薙刀式(v16)かな入力を行うためのソフトです。薙刀使いである弁慶から名前を採用しました。
完全に書き直したので、Benkei2と呼称することにします。

## 先代Benkeiからの主な変更点

1. QMK薙刀式のかな変換と同じように、かな変換候補が絞り込める時は積極的に確定していきます。
2. 先代Benkeiが、親指シフトのLacaille(kkadowaki氏作)をベースにしていたのに対し、Benkei2はSwiftでスクラッチから書き直したアプリケーションです。
3. カナ変換設定をファイルから読み込むようにしたので、ユーザーが変更可能です。初回の起動後に、`~/Library/Containers/jp.eswai.Benkei2/Data/config`に設定ファイルが作られるので、修正してください。

### 薙刀式の仕様

- [ ] ctrl+shift+0で薙刀式をオフにして通常の入力へ戻る（サスペンド）、
- [ ] ctrl+shift+1で再開。

- [x] IMEオフ（半角、英数直接入力モード）は、FG同時押し。
- [x] IMEオン（全角、カナ漢字変換モード）は、HJ同時押し。

- [x] ctrl, cmd, optを押している時はqwerty配列に戻る。
- [ ] SandS（シフトキーとスペースキーを兼ねる）を使用。
- [ ] MV同時押しにエンター。

- [x] 単打面、シフト面
- [x] その他同時押し面
- [x] 編集モード
- [x] 固有名詞モード
